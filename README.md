ИНСТРУКЦИЯ ПО УСТАНОВКЕ АДДОНА ДЛЯ ИГРОКОВ
═══════════════════════════════════════════════════════════════

1. УСТАНОВКА АДДОНА
───────────────────────────────────────────────────────────────

1.1. Скачайте архив с аддоном (client_side_addon.zip)

1.2. Распакуйте архив в папку addons вашей игры Garry's Mod

1.3. Путь установки:
    Windows: C:\Program Files (x86)\Steam\steamapps\common\GarrysMod\garrysmod\addons\
    Или: C:\Users\<ВашеИмя>\Desktop\GarrysMod\garrysmod\addons\
    
    Linux: ~/.steam/steam/steamapps/common/GarrysMod/garrysmod/addons/

1.4. Структура должна выглядеть так:
    garrysmod/addons/client_side_addon/
        ├── addon.txt
        └── lua/
            └── autorun/
                └── client/
                    └── cl_pirate_protection.lua


2. ПЕРВОЕ ПОДКЛЮЧЕНИЕ К СЕРВЕРУ
───────────────────────────────────────────────────────────────

2.1. Запустите Garry's Mod

2.2. Подключитесь к серверу: 26.211.58.48:27015
    (через консоль: connect 26.211.58.48:27015)

2.3. Если у вас пиратская копия игры:
    - Аддон автоматически определит это
    - Появится окно с загрузкой необходимых файлов
    - Дождитесь завершения загрузки
    - Игра автоматически переподключится к серверу

2.4. Если у вас лицензионная копия:
    - Ничего делать не нужно
    - Подключение происходит как обычно


3. ЗАГРУЗКА ФАЙЛОВ
───────────────────────────────────────────────────────────────

3.1. При первом подключении аддон автоматически:
    - Получит список необходимых файлов с сервера
    - Загрузит недостающие файлы
    - Сохранит их в папку addons

3.2. Если загрузка не началась автоматически:
    - Переподключитесь к серверу
    - Проверьте интернет-соединение

3.3. Если загрузка файла не удалась:
    - Появится окно с альтернативной ссылкой
    - Скачайте файл вручную по ссылке
    - Распакуйте архив в папку garrysmod/addons/
    - Переподключитесь к серверу


4. РАБОТА С ZIP АРХИВАМИ
───────────────────────────────────────────────────────────────

4.1. Если аддон загрузил ZIP архив:
    - Найдите файл в папке: garrysmod/addons/<имя_файла>/<имя_файла>.zip
    - Распакуйте ZIP архив в папку addons
    - Убедитесь, что структура правильная:
        garrysmod/addons/<имя_аддона>/
            └── lua/
                └── ...

4.2. После распаковки:
    - Перезапустите игру или выполните: lua_refresh в консоли
    - Подключитесь к серверу снова


5. ПОДКЛЮЧЕНИЕ К ДРУГИМ СЕРВЕРАМ
───────────────────────────────────────────────────────────────

5.1. Этот аддон работает ТОЛЬКО с сервером: 26.211.58.48:27015

5.2. При попытке подключиться к другому серверу:
    - Появится сообщение об эксклюзивности аддона
    - Аддон не будет работать на других серверах

5.3. Это нормальное поведение - аддон создан специально для этого сервера


6. РЕШЕНИЕ ПРОБЛЕМ
───────────────────────────────────────────────────────────────

6.1. Аддон не работает:
    - Проверьте, что файлы находятся в правильной папке addons
    - Убедитесь, что папка называется "client_side_addon"
    - Перезапустите игру

6.2. Файлы не загружаются:
    - Проверьте интернет-соединение
    - Проверьте, что сервер доступен
    - Попробуйте скачать файлы вручную по ссылке из сообщения об ошибке

6.3. Игра не подключается после загрузки:
    - Убедитесь, что все файлы распакованы правильно
    - Проверьте консоль игры на наличие ошибок (консоль: ~)
    - Попробуйте подключиться вручную: connect 26.211.58.48:27015

6.4. Ошибка "Неверная версия аддона":
    - Скачайте новую версию аддона
    - Удалите старую папку client_side_addon
    - Установите новую версию

6.5. Игра крашится:
    - Удалите аддон из папки addons
    - Проверьте целостность файлов игры
    - Свяжитесь с администратором сервера


7. ВАЖНЫЕ ЗАМЕЧАНИЯ
───────────────────────────────────────────────────────────────

7.1. Не изменяйте файлы аддона вручную

7.2. Не удаляйте аддон во время игры на сервере

7.3. При обновлении аддона:
    - Удалите старую версию
    - Установите новую версию
    - Перезапустите игру

7.4. Аддон автоматически определяет пиратскую копию
    - Это необходимо для загрузки контента
    - Информация используется только для работы аддона
